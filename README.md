# yunks128.github.io

Personal website for Kyongsik Yun, Ph.D. — migrated from Google Sites.

Static HTML/CSS, hosted on GitHub Pages at https://yunks128.github.io/.

## Local preview

Just open `index.html` in a browser, or:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy (GitHub Pages)

1. Push to `main` on `yunks128/yunks128.github.io`.
2. On GitHub: **Settings → Pages → Build and deployment** → Source: **Deploy from a branch**, Branch: `main`, folder: `/ (root)`.
3. Site publishes at https://yunks128.github.io/.
